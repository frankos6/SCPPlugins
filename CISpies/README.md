# CISpies

#### Adds CI spies to MTF waves

### Config
- SpyChance `int` - Chance to respawn a spy in a MTF wave (in percent)

  > default: `25`
- SpawnSpyOnce `bool` - Specifies if a spy should spawn only once per round

  > default: `true`

### Commands
- `.reveal` - Reveals the identity of the spy
